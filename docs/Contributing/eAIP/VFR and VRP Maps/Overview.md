# Overview

Congratulations! By now you have very likely begun coding your first eAIP. Wether it be Togo or FASA you may need to create 2 types of maps; Visual flight rules (VFR) and Visual reference points (VRP).

Furthermore, should you have any remaining questions, we strongly encourage you to ask in the VATSSA Community Discord server; a link can be found [here](https://community.vatsim.net/).
