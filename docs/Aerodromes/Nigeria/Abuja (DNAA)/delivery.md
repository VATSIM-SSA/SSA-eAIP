# Delivery

!!! info 
Abuja does not have a dedicated Delivery frequency, as such all Delivery responsibilities are assumed by Abuja Ground (DNAA_GND). Under no circumstances should a DNAA_DEL position ever be opened without prior approval from staff.

## Departure Procedures

Pilots shall be issued one of the RNAV departures when flying out of Abuja. 

In the event a pilot is unable to comply, a runway heading departure or other means should be issued to the pilot.

**Runway 04**

| Fix | Procedure | Remarks |
| :---------: | :---------: | :---------: |
| IKSEM | IKSEM1S | - | 
| IKVAD | IKVAD1S | - |
| OKORU | OKORU1S | - |

**Runway 22**

| Fix | Procedure | Remarks |
| :---------: | :---------: | :---------: |
| IKSEM | IKSEM1Q | - | 
| IKVAD | IKVAD1Q | - |
| OKORU | OKORU1Q | - |

## Initial Climb

The Initial Climb provided within the clearance will be set at FL110.

## Phraseology

SID

!!! info "Delivery Phraseology" 
"(callsign), cleared to (destination), via the (SID) departure, initial climb FL110, squawk
(squawk)"

Example
"ABCDE, cleared to Luanda, IKVAD1S departure, initial climb FL110, squawk 0532"
