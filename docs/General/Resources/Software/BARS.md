# BARS Plugin for FASA

## Overview
BARS is ane addon for Microsoft Flight Simulator that guides pilots using taxi lights to the runway or to a designated area on the airport.

## Installation

BARS is configurated in the sector file already, however you need an API link to connect to the BARS network and allow BARS to draw on the Ground radar. To allow bars to draw on the groundradar, you need to go to OTHER SET and plug-ins. This will open a window for the sector files plugins, you will find BARS in this list.
